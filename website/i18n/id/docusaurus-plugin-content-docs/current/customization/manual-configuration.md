# Costumization Landy Theme

Untuk kostumisasi tema landy Anda, Anda dapat mengikuti langkah:

1. Buka Tab Landy-tema, klik Konfigurasi
2. Anda dapat mengedit Judul Bilah Navigasi, URL Awalan, Tema Ikon Landy, dan Footer Tema Ikon Landy
3. Klik update config jika sudah selesai edit
